In-vox Invoice generator and Database Inventory

Concept for a small web sales managing system. The application uses different APIs to connect to ecommerce stores to search for articles and prices, then allows the user to add this articles directly to it's firebase realtime database.
The application allows the user to fill a form to create and send by email a PDF Invoice using the articles and prices stored in the Inventory database.
